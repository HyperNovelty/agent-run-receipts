# Code of Conduct

This project is intended for practical public-good tooling. Contributors are expected to communicate respectfully, focus on the work, and avoid exposing private information.

Do not use project spaces to share secrets, private traces, personal data, harassment, or material that creates unnecessary risk for maintainers or users.

When disagreement happens, keep the discussion specific to facts, tradeoffs, and user safety. Maintainers may remove contributions that are unsafe, irrelevant, abusive, or inconsistent with the local-only boundary.
