Agent Run Receipts

A local receipt records what an AI or coding agent was asked to do, what boundaries applied, what changed, what verification ran, and whether a human gate remains.

Public-Safety Boundary

This starter does not authorize public action. It does not provide legal advice, compliance certification, secret scanning, or private trace storage.

Try It Locally

Validate the example with python3 scripts/validate_receipt.py examples/agent-run-receipt.example.json.

Render HTML with python3 scripts/render_receipt_html.py examples/agent-run-receipt.example.json examples/rendered/agent-run-receipt.example.html.

Before sharing, review docs/publish-safety-reviewer-checklist.md.

Development Delta Receipts

When an AI or coding agent claims progress, add the optional development_delta section to show changed artifacts, capability improvement, validation evidence, next blocker, and preserved non-actions.

Read docs/development-delta-receipts.md and validate examples/development-delta-receipt.example.json.
